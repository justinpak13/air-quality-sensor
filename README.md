# Air Quality Sensor
An air quality sensor that displays the eC02 and TVOC level using the Rust Embassy crate. 

## Supplies Needed
- Raspberry Pi Pico
- CCS811 Air Quality Sensor
- OLED display using SPI
- LEDs (green, yellow, red)
- Jumper Wires
- Resistors for LEDs (I used 220 Ohms)
- Breadboard (Optional)

## Directions
Once all hardware is assembled, build the project. Then plug in the pico while holding the bootsel button and release. 

Then run cargo run
